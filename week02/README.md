# Week 2: Intermediate JavaScript & Mini-Projects

This folder contains scripts and mini-projects focusing on intermediate JavaScript concepts, object-oriented programming, and DOM manipulation.

## Files Overview

- **`class-demo.js`**: Demonstrates the use of ES6 Classes, constructors, and methods in JavaScript.
- **`date-hands-ons.js`**: Explores the `Date` object and various date/time manipulation techniques.
- **`deep-copy.js`**: Provides examples of performing deep copies of nested objects and arrays.
- **`shallow-copy.js`**: Covers shallow copying techniques (like the spread operator and `Object.assign()`).

## Sub-Projects

This week also includes several specialized sub-directories for hands-on projects:

- **`E-Commerce/`**: A mini e-commerce application structure.
- **`master-collections/`**: Exercises focusing on mastering advanced collections like Maps and Sets.
- **`todo-app/`**: A classic Todo List application demonstrating state management and UI updates.

## How to Run

For individual script files, run them using Node.js:

```bash
node class-demo.js
```
