# Week 3: Backend & Database Integration

This folder shifts focus towards backend development using Node.js and integrating with databases like MongoDB.

## Files Overview

- **`mongoDB-operations.txt`**: Contains essential MongoDB queries and operational commands for reference.

## Sub-Projects

- **`backend-demo/`**: Introductory examples and setup for building a backend server.
- **`backend+mongoDb/`**: Projects and scripts demonstrating the integration between a Node.js backend and a MongoDB database.
- **`E-commerce-backend/`**: The backend implementation and API routes for an E-commerce application.

## How to Use

Navigate into the sub-project directories for specific instructions on how to set up the environment, install dependencies, and start the backend servers.
