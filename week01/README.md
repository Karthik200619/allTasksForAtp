# Week 1: JavaScript Fundamentals

This folder contains hands-on scripts and exercises focusing on the core fundamentals of JavaScript.

## Files Overview

Here is a breakdown of the files included in this week's module:

- **`array-advance.js`**: Explores advanced array methods (like `map`, `filter`, `reduce`) and operations.
- **`array-simple.js`**: Covers basic array operations such as adding, removing, and iterating over array elements.
- **`control.js`**: Demonstrates control flow structures including `if-else` statements, `switch` cases, and loops.
- **`object-simple.js`**: Introduces JavaScript objects, properties, and basic object manipulation.
- **`operators.js`**: Contains examples of different JavaScript operators (arithmetic, logical, comparison).

## How to Run

You can run any of these files using Node.js. For example:

```bash
node array-simple.js
```
